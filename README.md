# Vanilla Forge 1.20.1

Минимальный репозиторий для лаунчера MCLaunch.

Что уже есть:
- манифест для Forge 1.20.1
- installer Forge
- папка `overrides` для будущих модов и конфигов

Куда класть файлы:
- моды: `overrides/mods`
- конфиги: `overrides/config`
- defaultconfigs: `overrides/defaultconfigs`
- ресурспаки: `overrides/resourcepacks`
- шейдеры: `overrides/shaderpacks`
